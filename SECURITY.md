# Security Policy / 安全政策

## Scope / 范围

This repository contains research software for backtesting. It does not provide live order routing, account custody, secrets management, or broker authentication.

本仓库包含回测研究软件，不提供实盘下单、账户托管、密钥管理或券商认证功能。

## Reporting / 报告方式

Please open a private security advisory or contact the maintainers if you find a vulnerability involving arbitrary code execution, unsafe file handling, dependency compromise, or accidental exposure of private data.

如果发现任意代码执行、不安全文件处理、依赖供应链风险或私有数据泄露问题，请通过私有安全公告或联系维护者报告。

## Data / 数据

Do not publish private account exports, API keys, broker credentials, or proprietary datasets unless you have explicit redistribution rights.

不要公开私人账户导出、API key、券商凭据或无再分发授权的专有数据。
