# Code Of Conduct / 行为准则

This project welcomes respectful research and engineering discussion.

本项目欢迎理性、尊重的研究和工程讨论。

## Expected Behavior / 期望行为

- Be respectful when discussing strategy results, risks, and limitations.
- Share reproducible evidence when making claims about performance.
- Respect data licenses and privacy.

- 讨论策略结果、风险和局限性时保持尊重。
- 对收益或性能做判断时提供可复现证据。
- 尊重数据授权和隐私。

## Unacceptable Behavior / 不接受的行为

- Harassment, insults, or discriminatory language.
- Presenting backtests as guaranteed future returns.
- Publishing private credentials, private account exports, or unauthorized datasets.

- 骚扰、侮辱或歧视性语言。
- 把回测结果包装成未来收益保证。
- 发布私人凭据、私人账户导出或未授权数据集。
